Random Walk on DA-IICT Campus Network

Empirical simulation vs. analytical stationary distribution

Problem Setup

We simulate a simple random walk on an undirected campus network. From any node i, the next node is chosen uniformly among its neighbors. We compare the empirical distribution of visits with the analytical stationary distribution.

Transition matrix P is row-stochastic: P[i,j] = 1/deg(i) if (i,j) is an edge, otherwise 0.

Analytical Result

For an undirected, connected, aperiodic graph, the stationary distribution π is unique and satisfies π = πP. Moreover, it has a simple closed form:

π(i) = deg(i) / (2|E|)

Intuition: nodes with more incident edges are visited more often in the long run.

In-app, we compute π via power iteration (robust and numerically stable), and verify it matches the degree-based formula.

Convergence

The simulation shows the Total Variation Distance (TVD) every few steps: 0.5 × Σ |π(i) − p_emp(i)|. As we run longer, TVD tends to 0.

You can point out the decreasing TVD in the dashboard as evidence of convergence.

How to Use During Presentation

  1. Open the dashboard at http://localhost:3000.
  2. Start the walk. Point out the Social Hub (most-visited node) and how it changes early on.
  3. Show the Comparison chart: empirical bars approach analytical bars.
  4. Highlight the TVD chart: the error trend decreases with time.

Tip: Slow the simulation to narrate; speed it up to demonstrate long-run behavior.
